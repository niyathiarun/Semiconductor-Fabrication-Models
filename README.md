# Semiconductor Process Simulation Portfolio  
### CMOS Fabrication Modeling using SEMulator3D

This repository contains a collection of semiconductor process simulations developed during an intensive 3-day hands-on workshop on SEMulator3D.

The projects focus on modeling critical  fabrication steps and advanced transistor architectures used in modern integrated circuit manufacturing. Through 3D process simulation, I explored how nanoscale device structures are formed layer-by-layer in a realistic fabrication flow.

---

## 🔬 Projects Included

### 1️⃣ Double Exposure Lithography
Simulation of multi-patterning technique used to achieve feature sizes beyond single exposure lithography limits.

### 2️⃣ STI (Shallow Trench Isolation)
Modeling trench etching, dielectric fill, and planarization steps used for device isolation in CMOS processes.

### 3️⃣ LOCOS Isolation
Simulation of local oxidation technique for field oxide formation and isolation in traditional CMOS technology.

### 4️⃣ FinFET Structure Modeling
3D modeling of advanced FinFET transistor architecture used in modern sub-20nm nodes.

---
